Nykaa Sales Analysis Dashboard (Power BI)

 Project Overview

This project is a Power BI dashboard created to analyze and visualize sales performance of Nykaa, an Indian e-commerce platform focused on beauty, wellness, and fashion products.
The dashboard provides actionable insights into sales trends, product performance, and customer behavior.

 Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Understand monthly and yearly sales trends
* Support data-driven business decisions



 Tools & Technologies

Microsoft Power BI
Power Query for data cleaning and transformation
DAX** for calculated measures and KPIs
Data Visualization** techniques



 Key Features

* Interactive dashboards and filters
* Sales trend analysis (time-based)
* Category-wise and product-wise sales comparison
* KPI cards for quick insights
* Clean and user-friendly visual design
